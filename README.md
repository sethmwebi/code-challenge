## Week One Code Challenge

#### Student Grade Generator

This piece of code generates grades according to a student's score  
It awards grades using the system below

| Marks Range | Grade |
| ----------- | ----- |
| > 79        | A     |
| 60 - 79     | B     |
| 50 - 59     | C     |
| 40 - 49     | D     |
| < 40        | E     |
